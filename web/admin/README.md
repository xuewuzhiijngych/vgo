# Vgo-Admin

### 介绍 📖

Vgo-Admin 一款基于 Geeker-Admin 一款基于 开源的后台管理框架.
配合Vgo使用

### Geeker-Admin文档 📚

- 文档地址：https://docs.spicyboy.cn

### 使用步骤

- **Clone：**

```text
# Gitee
git clone xxx
```

- **Install：**

```text
pnpm install
```

- **Run：**

```text
pnpm dev
pnpm serve
```

- **Build：**

```text
# 开发环境
pnpm build:dev

# 测试环境
pnpm build:test

# 生产环境
pnpm build:pro
```
